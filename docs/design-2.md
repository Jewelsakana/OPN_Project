### 当前任务  
实现TextEngine类的四个函数，要求如下： 
**注意**：  
1. 将TextEngine类的vector<stirng>改为引用传递，降低开销
2. 用换行符连接各行   
3. 参数中的行号与列号是从1开始的逻辑坐标，TextEngine内部需要将其转换为0开始的vector索引  
4. 同时要实现相应的测试用例来自动化测试  
**任务**：
1. append:追加文本，参数为字符串，要求在vector<string>末尾添加文本  
2. insert:在指定位置插入文本,参数为行号，列号，和插入的字符串，列号表示插入到第几个字符前。插入位置应在现有位置之前。同时插入文本中可以包含换行符('\n')，将自动拆分为多行.需要有异常处理，对行号或者列号越界抛出行号或列号越界异常，若为空字符串且不插入起始位置，则也需要抛出空字符串插入异常位置异常  
3. delete：在指定位置删除字符串，参数为行号，列号，和长度。删除范围不可以跨行，且删除长度不可以超过该行的剩余字符数。需要有异常处理，删除长度超出该行剩余字符，则对删除长度超出行尾抛出异常，若行号列号越界，则抛出相应的范围错误异常  
4. show:显示字符串；参数为startline和endline(包括该行在内)，应该返回一个拼接好的string。显示文本编辑器中指定的范围内容，参数默认应该为startline=0,endline=数组的末尾
